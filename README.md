# Home Builder — Unreal Engine 5 Project

Home Builder is a building‑game prototype developed in Unreal Engine 5.  
I am creating this project as part of my programming portfolio, documenting the development process and each stage of progress.

---

## 🎮 Prototype Features

- Advanced camera system (pan, zoom, rotate)
- Custom GameMode and PlayerController written in C++
- Terrain editor (sculpt + paint)
- UMG‑based UI (HUD, icons, panels)
- Enhanced Input system
- Custom C++ classes for camera, terrain, and HUD

---

## 🛠️ Technologies

- Unreal Engine 5.5.4
- C++ (Gameplay, Camera, Terrain, UI)
- Blueprints (rapid prototyping)
- UMG (user interface)
- Enhanced Input

## 🧩 Installation & Running the Project

To open and run the project locally:

1. Clone the repository:
git clone https://github.com/AdamSzukalski/Home-Builder.git
2. Open the project folder and right‑click `HomeBuilder.uproject`.
3. Select **Generate Visual Studio project files**.
4. Open the generated `.sln` file in Visual Studio or Rider.
5. Build the project (Development Editor).
6. Launch the Unreal Editor from the IDE or by double‑clicking the `.uproject` file.

The project requires **Unreal Engine 5.5.4** or newer.
